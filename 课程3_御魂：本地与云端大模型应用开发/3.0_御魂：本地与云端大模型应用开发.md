# 本地与云端大模型应用开发

## 课程简介
本课程聚焦于本地和云端大模型的应用开发，帮助学习者掌握如何利用各类大模型工具进行实际应用开发。从本地部署到云端调用，从基础应用到高级集成，全方位提升AI应用开发能力。

- 动手实践：每个模块都包含具体的实践任务
